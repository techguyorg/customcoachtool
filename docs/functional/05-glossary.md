# CustomCoachPro - Glossary

**Author:** Susheel Bhatt  
**Contact:** s.susheel9@gmail.com

---

| Term | Definition |
|------|------------|
| **Plan Assignment** | A workout or diet plan assigned to a client by their coach with start/end dates |
| **Coaching Request** | A request from a client to a coach to establish a coaching relationship |
| **Check-in** | A periodic progress report submitted by clients to their coach |
| **Workout Template** | A reusable workout program structure with weeks, days, and exercises |
| **Diet Plan** | A nutrition plan with daily macro targets and meal structures |
| **System Plan** | Content created by platform administrators, available to all users |
| **Custom Plan** | Content created by coaches for their specific clients |
| **Macros** | Macronutrients: protein, carbohydrates, and fat |
| **Adherence** | Percentage measure of how well a client followed their assigned plan |
| **Coach-Client Relationship** | The active connection between a coach and their client |
| **Progress Photo** | Photos uploaded by clients to visually track physical changes |
| **Measurement** | Body measurements recorded by clients (weight, waist, etc.) |
| **Periodization** | Workout programs with varying structure across different weeks |
| **RLS (Row Level Security)** | Database security that controls which users can access which data |
| **Edge Function** | Serverless backend function for custom logic |
