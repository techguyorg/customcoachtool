## Technology Stack

| Layer | Technology |
|-------|------------|
| Frontend | React 18, TypeScript, Vite |
| Styling | Tailwind CSS, shadcn/ui |
| State | TanStack Query |
| Backend | Supabase (PostgreSQL) |
| Storage | Azure Blob Storage |
| Email | Resend API |
| CI/CD | Azure DevOps |
