## Authentication
- Email/password sign up with role selection
- Email/password sign in
- Password reset via email
- Role-based access control
- Protected route navigation

## Dashboard (Coach)
- Stats cards (clients, pending requests, check-ins)
- Pending coaching requests
- Recent client check-ins
- Quick actions (invite client, create plan)

## Dashboard (Client)
- My coach card
- Active plans overview
- Progress snapshot
- Quick links to features

## Exercise Library
- Browse exercises with filters
- Filter by muscle group, equipment, difficulty
- View exercise details (instructions, tips, video)
- Mark exercises as favorites
- Create custom exercises (coaches)

## Workout Templates
- Browse workout programs
- Filter by type, difficulty, days/week
- View full program structure
- Start program (independent users)
- Create custom programs (coaches)
- Assign to clients (coaches)
- Clone existing templates

## Diet Plans
- Browse diet plans
- View plan details with meals and macros
- Create custom diet plans (coaches)
- Assign to clients (coaches)

## Recipe Library
- Browse recipes
- View ingredients and nutrition
- Create custom recipes

## Food Database
- Browse foods by category
- View nutritional information
- Create custom foods

## Client Management (Coach)
- View all clients with status
- Client detail view
- Add notes about clients
- Update client status
- Quick assign plans

## Check-in System
- Check-in templates
- Client check-in submission
- Adherence tracking
- Wellness metrics
- Photo attachments
- Coach review with feedback

## Progress Tracking
- Body measurements over time
- Weight progress charts
- Progress photos with poses
- Goals (create, track, complete)

## Nutrition Logging
- Daily meal logging
- Food/recipe search
- Macro tracking
- Daily totals

## Messaging
- Coach-client conversations
- Real-time updates
- Unread counts

## Notifications
- Real-time notification bell
- Various notification types
- Mark as read

## Coach Marketplace
- Browse available coaches
- Filter by specialization, rating
- View coach profiles
- Send coaching requests

## Favorites
- Favorite exercises, recipes, templates
- Favorites page for quick access

## Coach Settings
- Profile information
- Specializations
- Certifications
- Pricing
- Availability toggle
