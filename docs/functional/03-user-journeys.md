---

## Journey 1: Coach Registration

1. Visit landing page → Click "Get Started"
2. Select "I'm a Coach" role
3. Enter email, password, name
4. Account created → Redirected to coach dashboard
5. Complete coach profile (specializations, certifications, rates)
6. Ready to accept clients

---

## Journey 2: Client Registration

1. Visit landing page → Click "Get Started"
2. Select "I'm looking for fitness help" role
3. Enter email, password, name
4. Account created → Redirected to client dashboard
5. Complete onboarding wizard (stats, goals, restrictions)
6. Ready to use platform

---

## Journey 3: Client Finding a Coach

1. Client navigates to Coach Marketplace
2. Browse coaches with filters (specialization, rating, rate)
3. Click coach card to view full profile
4. Click "Request Coaching"
5. Enter personal message (optional)
6. Submit request → Status: Pending
7. Wait for coach response

---

## Journey 4: Coach Receiving Client

1. Coach sees notification of new request
2. Navigate to Requests page
3. Review client profile and message
4. Click "Accept" or "Decline"
5. If accepted: Client added to "My Clients"
6. Assign initial workout/diet plan
7. Send welcome message

---

## Journey 5: Weekly Check-in Cycle

### Client Side
1. Receive check-in reminder
2. Open check-in form
3. Enter adherence scores (workout, diet)
4. Enter wellness metrics (sleep, stress, energy)
5. Add measurements (optional)
6. Upload photos (optional)
7. Write notes about week
8. Submit check-in

### Coach Side
1. Receive notification of submission
2. Open check-in review
3. Review all submitted data
4. Write feedback
5. Rate check-in quality
6. Submit review
7. Client receives feedback notification

---

## Journey 6: Plan Assignment

1. Coach navigates to client details
2. Click "Assign Plan"
3. Select plan type (workout/diet)
4. Search and select plan
5. Set start date and duration
6. Add coach notes (optional)
7. Confirm assignment
8. Client receives notification

---

## Journey 7: Independent User Self-Service

1. Browse workout templates
2. Filter by type, difficulty
3. View template details
4. Click "Start Program"
5. Program added to active plans
6. Track workouts independently
7. Update measurements over time
