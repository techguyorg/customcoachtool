## Quick Start

### For New Developers

1. Read [Product Vision](./functional/01-product-vision.md) to understand the platform purpose
2. Review [User Personas](./functional/02-user-personas.md) to understand target users
3. Study [Architecture Overview](./architecture.md) for technical understanding
4. Follow [Deployment Guide](./deployment-guide.md) for local setup
5. Reference [Components Guide](./components-guide.md) and [Hooks Reference](./hooks-reference.md) during development

### For Business Analysts

1. Start with [Product Vision](./functional/01-product-vision.md)
2. Study [User Personas](./functional/02-user-personas.md) and [User Journeys](./functional/03-user-journeys.md)
3. Reference [Feature Catalog](./functional/04-feature-catalog.md) for requirements
4. Use [Glossary](./functional/05-glossary.md) for terminology

### For DevOps Engineers

1. Review [Architecture Overview](./architecture.md)
2. Follow [Deployment Guide](./deployment-guide.md) for CI/CD setup
3. Reference [Configuration Guide](./configuration-guide.md) for environment setup
4. Study [Security & RLS](./security-rls.md) for security policies

## Technology Stack Summary

| Layer | Technology |
|-------|------------|
| Frontend | React 18, TypeScript, Vite |
| Styling | Tailwind CSS, shadcn/ui |
| State Management | TanStack Query (React Query) |
| Routing | React Router v6 |
| Backend | Supabase (PostgreSQL + Auth + Edge Functions) |
| File Storage | Azure Blob Storage |
| Email Service | Resend API |
| CI/CD | Azure DevOps Pipelines |
| Hosting | Azure Static Web Apps |
