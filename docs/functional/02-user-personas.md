## Persona 1: Super Admin

**Role:** Platform Administrator

### Responsibilities
- Manage system-wide content (exercises, plans, foods)
- User oversight and moderation
- Platform configuration
- Analytics review

### Capabilities
- Full access to all platform features
- Create/edit/delete system content
- View all user data
- Manage platform settings

---

## Persona 2: Coach

**Role:** Fitness Professional

### Profile
- Personal trainers, nutrition coaches, online coaches
- Managing 5-50+ clients
- Need efficient tools for scaling business

### Responsibilities
- Create custom workout programs
- Create custom diet plans
- Monitor client progress
- Review client check-ins
- Communicate with clients

### Capabilities
- **Client Management:** View all clients, update status, add notes
- **Plan Creation:** Custom workouts and diet plans
- **Assignments:** Assign plans to clients with dates
- **Check-in Review:** Review submissions, provide feedback
- **Messaging:** Direct communication with clients
- **Analytics:** Business performance insights

### Dashboard View
- Pending coaching requests
- Recent client check-ins
- Client overview stats
- Quick actions

---

## Persona 3: Coached Client

**Role:** Client with Assigned Coach

### Profile
- Working with a fitness professional
- Receiving personalized plans
- Regular check-ins required

### Responsibilities
- Follow assigned workout plans
- Follow assigned diet plans
- Submit regular check-ins
- Update measurements and photos
- Communicate progress with coach

### Capabilities
- **View Plans:** Access assigned workout and diet plans
- **Track Workouts:** Log workout completion
- **Log Nutrition:** Daily meal tracking
- **Check-ins:** Submit weekly progress reports
- **Progress Tracking:** Measurements, photos, goals
- **Messaging:** Direct communication with coach

### Dashboard View
- Active plans overview
- My coach card
- Progress snapshot
- Upcoming check-in reminder

---

## Persona 4: Independent Client

**Role:** Self-Directed Fitness User

### Profile
- No assigned coach
- Self-motivated fitness enthusiast
- May be exploring coaches

### Responsibilities
- Self-manage fitness journey
- Track own progress
- Explore available resources

### Capabilities
- **Browse Content:** System workout templates, diet plans, exercises
- **Start Programs:** Self-assign workout programs
- **Track Progress:** Measurements, photos, goals
- **Log Nutrition:** Daily meal tracking
- **Find Coaches:** Browse marketplace, send requests
- **Favorites:** Save preferred content

### Dashboard View
- Quick start options
- Progress overview
- Featured content
- Find a coach CTA
